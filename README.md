# Spielliste

![見本](assets/spielliste.jpg)

"Spielliste' is a emulator management list.

When you write game information to a file, it is displayed.

# Requirement
 
* Python 3.7.5
* Pillow 8.2.0
* PySimpleGUI 4.39.1
 
# Installation
 
```bash
pip install -r requirements.txt
```
 
# Usage
 
```bash
git clone https://github.com/m-star18/Spielliste
cd Spielliste
python main.py
```
